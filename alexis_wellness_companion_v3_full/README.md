# Alexis Wellness Companion v3 Full

A cute pink and purple wellness companion made for Alexis.

## What's Included

- Medication checklist
- Daily cute quote
- Bible Verse of the Day
- Richard's Love Notes
- Water tracker
- Mood tracker
- Medication streak counter
- Confetti when meds are checked
- Self-Care Corner
- Cute feminine pink/purple design
- GitHub Pages ready
- PWA-ready so it can be added to an iPhone home screen

## Removed

- No 911 button
- No Call Richard button
- No emergency section
- No daily notes section

## How to Use

1. Open `index.html` in your browser.
2. Add/check off medication reminders.
3. Track water and mood.
4. Upload the folder to GitHub.
5. Enable GitHub Pages.

## GitHub Pages Setup

Settings → Pages → Deploy from branch → main → root.

## Important

This app stores data locally in the browser using localStorage. It is not a medical device and does not replace professional medical advice.
